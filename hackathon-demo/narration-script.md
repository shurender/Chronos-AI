Chronos Engine is an AI decision intelligence platform that turns your work history into simulated future paths.

Instead of asking for generic career or product advice, the user connects data from GitHub, Slack, Notion, uploads, or a safe demo workspace.

Chronos extracts that history into a memory graph, preserving source labels so demo data, uploaded files, and live external evidence stay distinct.

Here the user defines a concrete decision: whether to position a portfolio toward AI product engineering, full-stack engineering, or a broader project-focused path.

The simulation produces multiple evidence-backed timeline branches, showing expected regret, probability, milestones, and confidence breakdowns.

The Memory Graph makes the reasoning inspectable, so a judge can see which projects, decisions, sources, and evidence shaped the result.

Finally, the Future Self avatar answers a follow-up question using graph nodes and evidence citations instead of generic advice.

Chronos is built with React, FastAPI, Chroma, NetworkX, and provider-ready Fireworks, Tavily, and AMD/vLLM integrations.
