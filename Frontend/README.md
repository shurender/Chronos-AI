# Chronos Engine ⏳
**AMD Developer Hackathon: ACT II | Unicorn Track**

Chronos Engine is a Decision Intelligence digital twin platform. It ingests a user's past data to build a memory graph, then uses multi-agent parallel simulation to simulate thousands of future scenarios and optimize for "Expected Regret."

## 🚀 Tech Stack
* **Frontend Framework:** React 19 + Vite
* **Styling:** Tailwind CSS v4
* **State Management:** Zustand
* **Graph Visualization:** React Flow + Dagre (Auto-layout)
* **Icons:** Lucide React

## 📦 Installation Instructions for Teammates

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### 1. Clone the repository
Open your terminal and run:
```bash
git clone https://github.com/Gooofus/Chronos-Engine-UI
cd chronos-ui
```
### 2. Install dependencies

Because we are using Tailwind v4, make sure you run a fresh install:
```bash
npm install
```
### 3. Run the development server
```bash
npm run dev
```
### 4. View the App

Open your browser and navigate to: http://localhost:5173

---

*Note: This frontend currently runs on heavily structured mock data simulating a B2B Pivot decision. It is fully ready to be wired up to the backend Python APIs.*
